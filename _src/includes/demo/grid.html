Grid Example

Below is a demo of all of the grid classes available for use. Naming conventions were matched up with the Bootstrap names to provide ease of transition. However, there is (currently) only one column size, medium.

The medium column is the most used across Xavier.edu which is why it was selected.

There is also a .nowrap class that can be applied to a row to ensure the columns do not wrap (in process).

This demo uses background colors and borders to distinguish the various columns. The demo row also includes a margin that is not in real-time code. Gutters are built into the columns, there is not whitespce between columns.

Finally, there is a flexible .col column that does not take size parameters but will conform to the space and number of columns automatically. This would be best utilized in a 50/50 or 100% container column.
